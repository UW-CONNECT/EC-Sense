The Code for packet detection and moisture estimation is written in MATLAB. Autocorrelation fuction detects and demodulates the packet. Calaculate_Pathloss_1 function takes the time as input and output corresponding incident power and pathloss. Calculate_VWC_from_Pathlosses function takes in pathloss values as input and gives outputs the measured moisture value.

All the function required for moisture estimation are uploaded in this folder. Run Packet_Detection_Moisture_Estimation code.
